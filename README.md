# 🛒 Blinkit Sales Analysis — Power BI Dashboard

> An end-to-end business intelligence project analyzing Blinkit's sales performance, outlet metrics, and product trends using Microsoft Power BI.

---

## 📌 Overview

This project delivers an interactive Power BI dashboard for Blinkit (India's quick-commerce grocery platform). It follows a complete BI workflow — from raw data to actionable insights — enabling stakeholders to monitor sales KPIs, outlet performance, and item-level trends at a glance.

---

## 🪜 Project Steps

| Step | Description |
|------|-------------|
| 📋 Requirement Gathering | Defined business goals and KPIs |
| 🔍 Data Walkthrough | Explored raw data structure and fields |
| 🔗 Data Connection | Connected data source to Power BI |
| 🧹 Data Cleaning | Handled nulls, duplicates, and quality checks |
| 🧩 Data Modeling | Built relationships between tables |
| ⚙️ Data Processing | Transformed data using Power Query |
| 📐 DAX Calculations | Created measures for KPIs and metrics |
| 🎨 Dashboard Layout | Designed visual structure and UX |
| 📊 Charts Development | Built and formatted all visuals |
| 📁 Report Development | Assembled the final interactive report |
| 💡 Insights Generation | Derived actionable business findings |

---

## 📊 Key Metrics Analyzed

- Total Sales Revenue
- Number of Items Sold
- Average Sales per Outlet
- Outlet Size & Location Performance
- Item Type & Fat Content Breakdown
- Year-wise Outlet Establishment Trends

---

## 🛠️ Tools & Technologies

- **Microsoft Power BI Desktop**
- **Power Query** — data transformation
- **DAX** — calculated measures and KPIs
- **Excel / CSV** — source data

---

## 💡 Key Insights

- High-fat content items generate significantly more revenue
- Tier 3 cities outperform Tier 1 in total sales volume
- Supermarket Type 1 outlets contribute the highest sales share
- Fruits, Snack Foods, and Household items are top-selling categories
